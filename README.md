# customized-optimizers

Customized versions of existing optimizers.

## Installation

```bash
# With pip
pip install customized-optimizers

# With uv
uv pip install customized-optimizers
```

## Development

```bash
# Clone the repository
git clone https://github.com/67372a/customized-optimizers.git
cd customized-optimizers

# Install dependencies with uv
uv sync

# Run tests or scripts
uv run python -c "from customized_optimizers import OPTIMIZERS; print(OPTIMIZERS)"
```
